

野火【WiFi_AP6181】模块
============================


野火的STM32挑战者、Pro及RT1052 Pro系列开发板都板载了AP6181 WiFi模块，
该模块使用SDIO接口进行通讯，传输速率可达2MB/s，
野火为该模块在STM32、RT1052上配套了使用Lwip进行各种联网操作的应用，
包括mqtt、服务器、客户端以及网络摄像头的应用例子。
